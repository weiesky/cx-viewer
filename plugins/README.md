# Bundled Plugins

This directory ships with cx-viewer. It is the default location for built-in plugins
that are loaded at server startup by the plugin loader. Add `.js` or `.mjs` files here
to ship plugins with the package without requiring users to populate their personal
`~/.codex/cx-viewer/plugins/` directory.

For detailed documentation on the plugin system, see [docs/plugins.md](../docs/plugins.md).
