# TeamCreate

## Визначення

Створює нову команду для координації кількох агентів, що працюють над проєктом. Команди забезпечують паралельне виконання завдань через спільний список завдань та обмін повідомленнями між агентами.

## Параметри

| Параметр | Тип | Обов'язковий | Опис |
|----------|-----|--------------|------|
| `team_name` | string | Так | Назва нової команди |
| `description` | string | Ні | Опис / призначення команди |
| `agent_type` | string | Ні | Тип / роль керівника команди |

## Що створюється

- **Файл конфігурації команди**: `~/.claude/teams/{team-name}/config.json` — зберігає список учасників і метадані
- **Каталог списку завдань**: `~/.claude/tasks/{team-name}/` — спільний список завдань для всіх учасників команди

Команди мають відповідність 1:1 зі списками завдань.

## Робочий процес команди

1. **TeamCreate** — створити команду та її список завдань
2. **TaskCreate** — визначити завдання для команди
3. **Agent** (з `team_name` + `name`) — запустити учасників, які приєднуються до команди
4. **TaskUpdate** — призначити завдання учасникам через `owner`
5. Учасники виконують завдання та спілкуються через **SendMessage**
6. Завершити роботу учасників післ�� закінчення, потім **TeamDelete** для очищення

## Пов'язані інструменти

| Інструмент | Призначення |
|------------|-------------|
| `TeamDelete` | Видалити команду та каталоги завдань |
| `SendMessage` | Міжагентне спілкування всередині команди |
| `TaskCreate` / `TaskUpdate` / `TaskList` / `TaskGet` | Управління спільним списком завдань |
| `Agent` | Запустити учасників, які приєднуються до команди |
