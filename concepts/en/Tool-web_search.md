# web_search

`web_search` performs web search for text or images. The Codex manual describes `web_search` transcript items whenever Codex searches the web; the capability can also be configured as cached, live, or disabled.

Use it when:

- The user explicitly asks to browse, search, or verify current information.
- The answer needs official documentation, direct links, or precise source attribution.
- The relevant facts may have changed.

Response-side `web_search_call` entries map to this document.
