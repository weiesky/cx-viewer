# Read

## 定义

从本地文件系统读取内容。根据当前 Codex 工具体系，可能支持文本文件、图片、PDF 和 notebook。

## 参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `file_path` | string | 是 | 文件绝对路径 |
| `offset` | number | 否 | 分段读取的起始行 |
| `limit` | number | 否 | 读取行数 |
| `pages` | string | 否 | 支持 PDF 时使用的页码范围 |

## 使用场景

**适合使用：**
- 读取已知源码文件和配置文件
- 查看用户提供的图片或截图
- 编辑或总结文档前读取内容
- 并行加载多个相关文件

**不适合使用：**
- 读取目录列表，应使用 shell/list 类工具
- 开放式仓库探索，应使用搜索或 SubAgent

## 注意事项

- 绝对路径在 CX Viewer 中最容易审计。
- 大文件通常分段读取。
- 图片和文档处理能力取决于当前 Codex runtime/tool surface。
