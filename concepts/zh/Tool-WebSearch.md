# WebSearch

## 定义

执行网页搜索，返回带来源的信息，用于获取当前事实。

## 参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `query` | string | 是 | 搜索查询 |
| `allowed_domains` | string[] | 否 | 仅包含这些域名 |
| `blocked_domains` | string[] | 否 | 排除这些域名 |

## 使用场景

**适合使用：**
- 当前事件和近期变化的事实
- 最新产品、包、API 或政策信息
- 为技术问题查找一手文档

## 注意事项

- 如果搜索结果影响了回答，应在最终回答中引用来源。
- 技术、法律、金融、医疗等主题优先使用一手来源。
- CX Viewer 会把 web-search 工具事件与 MainAgent/SubAgent 请求体分开记录。
